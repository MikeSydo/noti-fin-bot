# Security Policy

## Supported Versions

We currently provide security updates only for the latest version available in the `master` branch.

| Version | Supported          |
| ------- | ------------------ |
| 0.2.x   | :white_check_mark: |
| < 0.2.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please **do not** open a public issue. This helps protect the users of the bot.

Instead, please report it privately by sending an email to:
**mikesydocompan@gmail.com**

We appreciate your help in keeping NotiFinBot secure. We will acknowledge your report and work on a fix as soon as possible.
